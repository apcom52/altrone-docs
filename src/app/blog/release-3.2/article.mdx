  # More easy way to build human-friendly tables

  Now you can specify the type of the column in the `DataTableColumn` interface. This property is used to determine the type of the column and to apply the appropriate filter.
  For example, if you set the type to `date`, the filter will be a date picker. If you set the type to `number`, the filter will be a number input and etc.
  Also this property is used to determine the default renderer for the column. If you use `currency` type, the number will be formatted as a currency.

</Text.Section>

<Text.Section gap="l">
  # Migration to v3.2

  Altrone 3.2 is fully compatible with Altrone 3.0 and Altrone 3.1. But we recommend you to make some updates in your code
  to use new features and improvements.

  - If you use `DataTable` component, we highly recommend you to set the `type` property to the `DataTableColumn` interface.
  It will improve filtering, sorting and rendering of the column.

  - Also don't forget to set `locale` prop to provide better rendering of the date and number values.

  - Old `Component` prop of `DataTableColumn` interface is marked as deprecated. We recommend to use new `renderFunc` property instead.
</Text.Section>
