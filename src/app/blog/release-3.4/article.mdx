  This new component allows users to work with color in two modes - selecting an arbitrary color from the RGB color palette 
  or selecting one of the preset colors. Developers can limit the modes if they wish, allowing them to work with only one of them

<Text.Section gap="l">
  # Badges are back!

  In the old version of Altrone, we presented indicators for some components. With the upgrade to version 3 this feature was lost. 
  However, in this update we have brought back the ability to add indicators for many components (such as buttons, links in navigation, tabs, etc)
  using a new property - `badge`. However, you cannot set the location of the badge, but now you can send text to it in addition to numbers!
</Text.Section>
<Text.Section gap="l">
  # Refreshed UI for Tabs and Messages

  With each update, we make slight design changes to some components, making them more user-friendly and attractive to users. 
  This time, the turn came for `Tabs` and `Message`. `Tabs` got an updated design that is more versatile for your interface, and 
  the new `Message` design became more accessible, getting improved colors, a close button, and a compact mode.  
</Text.Section>

<Text.Section gap="l">
  # Migration to v3.4

  Altrone 3.4 is compatible with Altrone 3.3. 
  
  But there are some UI changes with Message and Tabs components. If you used custom styles for these components, please check them after the upgrade.

  Also we marked `Component` property as deprecated in `Select` component. Please use `renderFunc` property instead. This property will be removed in the next major version.
