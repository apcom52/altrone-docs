  # Usage

  This class provides a set of static methods with global utilities. You don't need to create an instance of this class.

  <CodeBlock children={`import { GlobalUtils } from "altrone-ui"`} />
  </Text.Section>

  <Text.Section gap="l">
    # Methods

    <CodeBlock children={`static uuid() => string`} />
    Returns unique string id

    <CodeBlock children={`static formatConsoleMessage(message: string) => string`} />
    Returns message for console with formatted substrings. Wrap some pieces of message in [[...]] braces to make them bold.

    <CodeBlock children={`static deprecatedMessage(component: string, deprecatedProp: string, instead: string, removedIn: string) => void`} />
    Shows console warning message about deprecated prop in component.

    <CodeBlock children={`static isTestEnvironment() => boolean`} />
    Returns `true` if the current environment is a test environment (it checks special global variable in Vitest).

    <CodeBlock children={`static getNumberDelimitersByLocale(locale: string) => { decimal: string, grouping: string }`} />
    <NewLabel version={32} />
    Returns decimal and grouping delimiters for the specified locale.
  </Text.Section>
