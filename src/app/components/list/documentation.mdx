  This component is used to display a series of content. List supports gaps and separators between item, you can also hide some of the items and set a special item render function.

  # Rendering an item

  This component has a `renderItem` property that allows you to specify a render function for your data object. This function takes a parameter called `context`, which provides all the necessary information about the current item, such as its index and the item itself.

  You can also set a separator between items (this separator is not rendered for the last item) via `SeparatorComponent` prop.

  By default List component uses index as key for list rendering. If you want to change this behavior you need to provide your own `keyExtractor` function.
