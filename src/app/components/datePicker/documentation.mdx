# Usage

  Use `<DatePicker>` component to render date picker. To render month picker, year picker or date range picker use one of
  the subcomponents (like `<DatePicker.MonthPicker>` or `<DatePicker.RangePicker>`).


  Pay attention that Altrone uses `day.js` library and `Dayjs` objects instead of standard `Date` objects.

  <Playground code={`import { Form, Select } from 'altrone-ui';

const OPTIONS = [
  { label: 'Afghanistan', value: 'Afghanistan' },
  { label: 'Albania', value: 'Albania' },
  { label: 'Algeria', value: 'Algeria' },
  ...
]

const [country, setCountry] = useState<string>();
const [visitedCountries, setVisitedCountries] = useState<string[]>([]);

return <Form>
  <Form.Field label="Your Motherland">
    <Select value={country} placeholder="Where have you born?" onChange={setCountry} options={OPTIONS} />
  </Form.Field>
  <Form.Field label="Visited Countries">
    <Select value={visitedCountries} multiple placeholder="Where have you born?" onChange={setVisitedCountries} options={OPTIONS} />
  </Form.Field>
</Form>
`}>
      <DatePickerDemo />
    </Playground>

<Text.Section>
  # Year Picker

  This is subcomponent (you **don't** need to wrap it into `<DatePicker>`) where you can choose a single year (in this case date and month are always the first day of the
  first month of the selected year)

  <PropertiesTable title="Properties" data={DatePickerProperties} />
</Text.Section>

<Text.Section>
  # Range Picker

  This is subcomponent (you **don't** need to wrap it into `<DatePicker>`) where you can choose a two dates year (at first you need to choose start date and then you can choose end date)

  <PropertiesTable title="Properties" data={DatePickerRangeProperties} />
