# Usage

This component is available only in Altrone 3.3 and above.

In Altrone we use 12 column grid system. When you set `size` prop to `12` it will take 100% of the width.
If total sum of all columns is more than 12, it will be wrapped into additional row. You can also create gaps
between columns by using `offset` prop.

Using this component you can create complex layouts with ease.

  <Playground code={`import { Grid } from 'altrone-ui';

return <Grid>
  <Grid.Column size={12}>Header</Grid.Column>
  <Grid.Column size={4}>Sidebar</Grid.Column>
  <Grid.Column size={8}>Content</Grid.Column>
  <Grid.Column size={12}>Footer</Grid.Column>
</Grid>`}>
<Grid>
  <Grid.Column size={12}>
    <div className={s.GridCell}>Header</div>
  </Grid.Column>
  <Grid.Column size={4}>
    <div className={s.GridCell}>Sidebar</div>
  </Grid.Column>
  <Grid.Column size={8}>
    <div className={s.GridCell}>Content</div>
  </Grid.Column>
  <Grid.Column size={12}>
    <div className={s.GridCell}>Footer</div>
  </Grid.Column>
</Grid>
</Playground>
