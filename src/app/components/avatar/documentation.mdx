# Usage

The purpose of this component is very simple - to display user avatar (it could be photo or initials). When user has no photo, initials are displayed.

By default when initials are displayed, the component shows first letter of the first name and last name. But last name is optional, so when it is not provided, only first letter of the first name will be displayed.
