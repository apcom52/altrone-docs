# Usage

  `<Popover>` is a great and convenient way to create a dropdown on the page. It requires an element around which positioning will occur.

  You need to wrap your interactive element into a popover. It can be button, link or any other interactive element. When user clicks on
  the wrapped element (you can also set type of interaction: `click`, `hover` or `focus`) the popover is shown on the page.

  <Playground code={`import { Popover, Flex, Button, Icon, Text } from 'altrone-ui';

<Popover title="Delete confirmation" showCloseButton content={({ closePopup }) => (
  <Flex direction="vertical" gap="m">
    <Text.Paragraph>
      Do you really want to delete the item?
    </Text.Paragraph>
    <Flex direction="horizontal" justify="end" gap="m">
      <Button onClick={closePopup} label="Cancel" />
      <Button onClick={closePopup} role="danger" label="Delete" />
    </Flex>
  </Flex>
)}>
  <Button leftIcon={<Icon i="delete" />} label="Delete item" />
</Popover>
`}>
      <Popover title="Delete confirmation" showCloseButton content={({ closePopup }) => <Flex direction="vertical" gap="m">
        <Text.Paragraph>
          Do you really want to delete the item?
        </Text.Paragraph>
        <Flex direction="horizontal" justify="end" gap="m">
          <Button onClick={closePopup} label="Cancel" />
          <Button onClick={closePopup} role="danger" label="Delete" />
        </Flex>
      </Flex>}>
        <Button leftIcon={<Icon i="delete" />} label="Delete item" />
      </Popover>
    </Playground>

# Position of the popover

By default, Altrone uses auto-positioning for Popovers and choose more suitable position depends on interactive element position
and scroll position of the page. By you can set a specific position using `placement` prop.

When you set the placement of the popover but there is no enough space for displaying, the popover is displayed in the opposite side.
</Text.Section>
