  # Usage

  To use this component you need to import `useToast` hook. This hook returns 4 functions: `toast()`, `success()`, `warning()` and `danger()`.
  Each function is responsible for every type of notification.

  <Playground code={`import { useToast, Flex, Button } from "altrone-ui";

const { toast, success, warning, danger } = useToast();

return <Flex gap="s">
  <Button
    label="Show toast message"
    onClick={() => toast('New updates are available.')}
  />
  <Button
    label="Successful toast"
    onClick={() => success('File uploaded successfully!')}
  />
  <Button
    label="Warning toast"
    onClick={() =>
      warning('Unsaved changes will be lost. Do you want to proceed?')
    }
  />
  <Button
    label="Danger toast"
    onClick={() => danger('An error occurred. Please try again')}
  />
</Flex>
`}>
    <ToastsDemo />
    </Playground>
</Text.Section>
