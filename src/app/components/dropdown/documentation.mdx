  # Usage

  `<Dropdown>` is a kind of Popover that aims to create context menus in application interfaces. This component provides
  some useful features to build accessible and powerful menus.

  Also this component has some subcomponents which are menu items that fulfil different actions (it can be simple action,
  checkbox, radio list or child menu).
