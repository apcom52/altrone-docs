# Usage

  Using this component you can build input with autocomplete. It is useful when you need to provide a list of suggestions to the user.

  <Playground code={`import { AutocompleteInput, Form } from 'altrone-ui';

const [country, setCountry] = useState('');

const getCountries = () => {
  return COUNTRIES.filter(c => c.toLowerCase().startsWith(country.toLowerCase()));
}

return <Form>
  <Form.Field required name="country" label="Your country">
    <AutocompleteInput getSuggestions={getCountries} placeholder="Enter your country" value={country} onChange={setCountry} />
  </Form.Field>
</Form>
`}>
